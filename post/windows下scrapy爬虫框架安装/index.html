windows下scrapy爬虫框架安装

2017-01-19

文章目录

1、安装python * 根据你的需求下载python安装包，安装python（本文基于python27）(https://www.python.org/downloads/) * 在 环境变量—“Path”中加入路径：”安装目录/python27/; 安装目录/python27/Scripts;（pip的路径）” * 在运行中输入”cmd”打开命令窗口，输入”python –version”，如果成功显示python版本号，则python安装成功！

2、安装pywin32 * 下载pywin32安装包,注意和python位数匹配，而不是和系统 (https://sourceforge.net/projects/pywin32/files/pywin32/Build%20217/)

3、安装pip * 一般为python默认安装好的，在”安装目录/python27/Scripts”目录中有”pip.exe”则默认安装成功； * 若python未默认安装，在 (https://pip.pypa.io/en/latest/installing/) 下载”get-pip.py”的python程序， * 在运行中输入”cmd”打开命令窗口，输入”python get-pip.py”即可自动安装。 * 重启cmd命令窗口，输入”pip –version”，若显示pip版本号，则安装成功！

4、安装Microsoft Visual C++ Compiler for Python 2.7 （这是比较重要的依赖软件） 5、安装twisted * pip install Twisted 这是比较推荐的安装方法

6、安装pyOpenSSL * 从这里(https://pypi.python.org/pypi/pyOpenSSL)下载，注意要选择对应python的版本。

7、安装lxml * 在(https://pypi.python.org/pypi/lxml/3.6.0#downloads) 中下载lxml for 2.7，下载安装。

8、安装scrapy pip install scrapy 若成功则会出现一开始的图片。

windows上的安装确实有点复杂，在linux就会简单很多。
